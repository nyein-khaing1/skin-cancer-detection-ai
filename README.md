# Skin Cancer Detection with Machine Learning

This project develops a machine learning system to detect skin cancer using the ISIC 2024 dataset.

## Technologies

* Python
* Scikit-learn
* TensorFlow
* NumPy
* Pandas
* Matplotlib

## Models Used

* Random Forest
* Logistic Regression
* CatBoost
* XGBoost

## Features

* Data preprocessing and feature engineering
* Handling class imbalance
* Model evaluation using pAUC
* Data visualisation and analysis

## Dataset

ISIC 2024 Skin Cancer Detection Dataset from Kaggle.
